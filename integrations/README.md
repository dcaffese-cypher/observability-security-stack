# Optional integrations

Playbooks and configs for **Zabbix** and **Wazuh** live here. They are **independent** of the main observability stack under `observability/`.

- **zabbix/** — Agent install templates; replace inventory placeholders before use.
- **Wazuh/** — Example RedHat playbook; sanitize `inventory.ini`.

These are starting points only; validate against your security baseline before production use.
